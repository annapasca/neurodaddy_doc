Power pipelineÂ¶

The power pipeline computes the power spectral density (PSD) on epochs or raw data on sensor space.

The input data has to be in fif format.

The outputs are the psd tensor and frequencies in .npz format stored in the workflow directory.

The power pipeline is implemented by the function create_pipeline_power and its Node power wraps the MNE Python functions mne.time_frequency.psd_welch, mne.time_frequency.psd_multitaper computing the PSD using Welch’s method and multitapers respectively.

See PSD pipeline to get an example on how to write a power pipeline.
