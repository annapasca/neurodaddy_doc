CINQ
====

See :ref:`example` section


Download scripts
----------------

.. code-block:: bash

    git clone https://github.com/davidmeunier79/neuropype_workshop.git
    cd source_code

Upgrade neuropype_ephy
----------------------
 
Go to the directory where you installed neuropype_ephy 
 
.. code-block:: bash

    git checkout dev
    git pull origin dev
    
    
Preprocessing 
-------------

This analysis is based on :ref:`preproc_meeg` Section

* set ``main_path`` in ``params_ica.py`` file to the folder where you downloaded the sample datasets
* look at the variables you can set
* run the script by the command 

.. code-block:: bash
   
    python run_preprocess_pipeline.py

* open ``ipynb_report.ipynb`` running a jupyter notebook

.. code-block:: bash
    
    jupyter notebook 
    
* look at the workflow directory
* inspect each report file and save the raw data file in the subject directory
